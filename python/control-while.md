# while
while문은 제어문에서 반복문이다.  
어떠한 조건 안에 반복하면서 데이터 값을 처리한다.

### **기본형식**
```py
while (변수) (조건):
  (조건 동안 반복하면서 처리할 동작)
```
간단한 예를 보면
```py
v1 = 1
while v1 < 11:
  print('v1 is : ', v1)
  v1 += 1
```
```
v1 is :  1
v1 is :  2
v1 is :  3
v1 is :  4
v1 is :  5
v1 is :  6
v1 is :  7
v1 is :  8
v1 is :  9
v1 is :  10
```
